Detector de IA: AI-writing signal review workflow
=====================================================

Overview
--------

Detector de IA is a Spanish-first bilingual web tool for reviewing pasted text or supported documents with probabilistic AI-likelihood signals and sentence highlights.

This documentation project is a compact workflow reference. It explains what the product accepts, what the report returns, how to interpret sentence-level context, and where the limits are before using the canonical source link.

Workflow
--------

- Paste text or review extracted text from a supported document.
- Run an AI-likelihood check.
- Inspect scores, evidence strength, reliability notes, and sentence highlights.
- Copy the summary or export a printable report for manual review.

Use cases
---------

- Students reviewing a draft before deciding which passages need manual revision.
- Teachers and academic reviewers using detector output as one signal among broader writing evidence.
- Editors and content teams checking whether draft language looks overly uniform or AI-styled.
- Researchers and multilingual reviewers comparing AI-likelihood signals across text or document excerpts.

Limits
------

The report is probabilistic and can include false positives or false negatives. Pasted text must be 300 to 100,000 characters, files must stay under 12 MB, and the result should support human review instead of deciding authorship or misconduct by itself.

Public wording should stay narrow: avoid claims about perfect accuracy, guaranteed detection, unlimited use, local-only processing, zero retention, official AI-platform affiliation, or definite authorship conclusions.
